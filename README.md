# mac-work-env
## NVCHAD (neovim framework)
You can see nvchad basic documentation on [here](https://nvchad.com/docs/quickstart/install)
Copy this config to custom lua
```shell
cp -r ./nvchad-custom/* ~/.config/nvim/lua/custom
```
